## 核心观点

复位启动不是"拉高就完事"，而是涉及**多物理域、多时间尺度、多状态交互**的复杂过程。

## 案例：持续 3 个月的"随机启动"谜团

2021 年某智能门锁芯片：冬季约 0.1% 设备随机启动失败，带回实验室后无法复现。

**关键模式**：故障只发生在老旧小区深夜，与交流电电压波动相关。

**连锁反应**：
1. 老旧小区深夜电压波动 ±15%，开关电源输出 50-100mV 纹波
2. 冬季低温使复位检测电路迟滞窗口从 50mV 收缩至 30mV
3. 电源纹波恰好在 CPU 核心电压达到复位阈值时出现，复位信号在 5μs 内反复跳变 3 次
4. A53 启动状态机在快速复位脉冲下进入未定义状态
5. 反复复位导致内部 ROM 读取指针溢出，破坏启动代码

**根源**：复位检测电路温度补偿不足 + 电源监控器响应过慢。代价：启动时间增加 2ms。

## 复位启动的六个时间尺度

| 尺度 | 范围 | 关键事件 |
|------|------|----------|
| 纳秒 | 时钟周期 | 时钟稳定、复位同步器亚稳态窗口 |
| 微秒 | PLL 锁定 | PLL 锁定、内存控制器初始化 |
| 毫秒 | DDR 训练 | DDR 训练、Flash 初始化 |
| 秒 | OS 引导 | 操作系统引导、应用程序加载 |
| 分钟 | 温度稳定 | 温度稳定、电源完全建立 |
| 年 | 器件老化 | 器件老化、数据保持期限 |

## 五个物理域交互

1. **电气域**：电源斜坡、噪声、纹波、毛刺
2. **热域**：温度梯度、自加热效应、热阻
3. **时钟域**：多时钟同步、相位对齐、抖动
4. **逻辑域**：状态机、计数器、有限状态机
5. **存储域**：寄存器初始化、内存内容、非易失存储

## A53 复位系统的复杂性维度

### 层次复杂性（7 种复位类型）

| 复位类型 | 影响范围 |
|----------|----------|
| 上电复位 PORESET | 所有逻辑，包括模拟模块 |
| 系统复位 SYSRESET | 数字逻辑，保持内存内容 |
| 调试复位 DBGRESET | 仅调试子系统 |
| 核心复位 CORERESET | 复位 CPU 核心，保持缓存内容 |
| 软件复位 | 通过系统寄存器触发 |
| 看门狗复位 | 超时触发 |
| 热复位 | 温度超过阈值触发 |

### 空间复杂性
```
复位源 → 复位分配网络 → 时钟域同步器 → 功能模块
    ↑         ↓              ↓              ↓
电源管理  时钟管理        状态保持      逻辑初始化
```

### 时间复杂性
```
T0：电源稳定达到 90%
T1 = T0 + 100μs：模拟模块初始化完成
T2 = T1 + 50μs：时钟稳定，PLL 锁定
T3 = T2 + 10μs：复位同步器稳定
T4 = T3 + 1μs：核心复位释放
T5 = T4 + 100 周期：第一条指令执行
```

## 复位检测电路的精妙设计

### 关键参数
- 上升阈值 V_rise：1.0V ±5%
- 下降阈值 V_fall：0.9V ±5%
- 迟滞电压 V_hyst：100mV
- 滤波时间 T_filter：10μs
- 最小复位脉宽 T_min：1μs

### 三级电源噪声抑制
1. **模拟滤波**：RC 低通，截止频率 1MHz
2. **数字滤波**：移动平均，窗口 8 个采样
3. **时序滤波**：最小脉宽要求，忽略 <1μs 脉冲

### 温度补偿机制
- 斩波稳定比较器，降低失调电压温度敏感性
- 温度补偿电阻网络
- 数字温度传感器校准，动态调整阈值

## 复位分配网络

**星型+树型混合拓扑**：
```
复位源 → 中央缓冲器 → 区域缓冲器 → 本地缓冲器 → 寄存器复位端
    ↑         ↑            ↑            ↑
时钟域A   时钟域B      时钟域C      时钟域D
```

**时序约束**：
- 时钟域内偏移：<100ps
- 时钟域间对齐：<2 个目标时钟周期
- 核心复位比外设复位早释放 10 个周期
- 时钟稳定后至少保持 5 个周期

## 电源序列与复位相互作用

**理想序列**：常电域(AON) → 核心域(CORE) → 内存域(MEM) → 外设域(PERIPH)

**分阶段交错上电**：
- 阶段 1：AON 上电，复位控制器工作
- 阶段 2：MEM 上电，初始化保持寄存器
- 阶段 3：CORE 上电，执行启动 ROM
- 阶段 4：PERIPH 上电，由软件控制

**电压斜坡率要求**：0.5~5V/ms，单调递增
