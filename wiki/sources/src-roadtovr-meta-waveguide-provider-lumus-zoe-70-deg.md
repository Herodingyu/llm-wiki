# Meta Waveguide Provider Claims "world's first" 70° FoV Waveguide

## 来源

- **原始文件**: raw/industry/smart-glasses/roadtovr-meta-waveguide-provider-lumus-zoe-70-deg.md
- **提取日期**: 2026-05-02

## 摘要

Lumus, the waveguide provider for Meta's Ray-Ban Display glasses, announced a 70° field-of-view ZOE waveguide at CES 2026. This matches the FoV of Meta's Orion prototype but achieves it using conventional glass waveguide processes rather than expensive silicon carbide, potentially enabling scalable manufacturing.

## 关键要点

- Lumus achieved 70° FoV in glass waveguides, matching Meta Orion's SiC-based performance
- Claims it as "world's first geometric waveguide to surpass a 70° FOV"
- Uses same manufacturing process as Ray-Ban Display waveguides (already proven at scale)
- ZOE optic shown at CES 2026; prototypes typically don't include battery/compute hardware
- Likely on the shortlist for Meta's first wide FoV AR glasses (targeted before 2030)

## 产品/技术细节

- **Lumus ZOE**: 70° field-of-view geometric waveguide
- **Material**: Conventional glass (not silicon carbide)
- **Manufacturing**: Same proven process as Ray-Ban Display waveguides
- **Tradeoffs**: Unknown impact on brightness, PPD, and visual artifacts at this FoV
