# QD-OLED

## 定义

QD-OLED（Quantum Dot OLED，量子点 OLED）是三星显示（Samsung Display）开发的一种 OLED 显示技术。与传统 WOLED 使用白色 OLED 配合彩色滤光片不同，QD-OLED 采用蓝色 OLED 作为光源，通过量子点（Quantum Dot）层将部分蓝光转换为红光和绿光，从而实现更高的色纯度和亮度。

## 技术细节

工作原理：

- **蓝色 OLED 光源**：底层为蓝色 OLED 发光层
- **量子点色转换层**：
  - 红色量子点：吸收蓝光，发射窄光谱红光
  - 绿色量子点：吸收蓝光，发射窄光谱绿光
  - 蓝色像素：直接使用蓝色 OLED 光（无色转换）
- **无彩色滤光片**：量子点自身即实现颜色选择，光损失更小

技术优势：
- **更高色域**：量子点发射光谱极窄，NTSC 色域可达 120%+
- **更高亮度**：省去彩色滤光片的光损失，光提取效率更高
- **更纯色彩**：窄发射光谱使色彩更饱和
- **更广视角**：量子点层对视角不敏感

与 WOLED 的对比：
| 特性 | QD-OLED | WOLED |
|------|---------|-------|
| 光源 | 蓝色 OLED | 白色 OLED |
| 色转换 | 量子点 | 彩色滤光片 |
| 色域 | 更宽 | 宽 |
| 亮度 | 更高 | 高 |
| 寿命 | 类似 | 类似 |
| 供应商 | 三星显示 | LG Display |

产品应用：
- 三星 S95C/S90C 系列电视
- 索尼 A95L 系列电视
- Dell/三星高端显示器

挑战：
- 量子点材料的长期稳定性
- 制造工艺复杂度高于 WOLED
- 生产成本较高
