# Meta Waveguide Provider Claims "world's first" 70° FoV Waveguide

## 来源

- **原始文件**: raw/industry/smart-glasses/roadtovr-meta-waveguide-provider-lumus-zoe-70-deg.md
- **提取日期**: 2026-05-02

## 摘要

Lumus作为Meta Ray-Ban Display智能眼镜的波导供应商，在CES 2026上展示了其革命性的ZOE波导技术，实现了70°的视场角（Field of View）。这一突破具有里程碑意义，因为它在常规玻璃波导工艺上达到了与Meta Orion原型机（采用昂贵的碳化硅SiC材料）相当的视场角性能，但生产成本和制造复杂度大幅降低。

传统的几何光波导技术长期受限于较窄的视场角（通常在30-50°范围内），而Lumus ZOE将这一界限一举突破至70°，被Lumus宣称是"全球首款视场角超过70°的几何光波导"。更关键的是，ZOE采用与Ray-Ban Display波导相同的成熟制造工艺，这意味着该技术已经具备了规模化量产的能力，不需要开发全新的生产线。这一技术进展使Lumus很可能进入Meta首款宽视场角消费级AR眼镜（目标在2030年前推出）的供应商短名单。

## 关键要点

### ZOE波导核心技术参数

| 参数 | 规格 |
|------|------|
| **视场角 (FoV)** | 70° |
| **材料** | 常规玻璃（非碳化硅SiC） |
| **波导类型** | 几何光波导 (Geometric Waveguide) |
| **制造工艺** | 与Ray-Ban Display波导相同的成熟工艺 |
| **展示时间** | CES 2026 |

### 技术突破意义

- **成本优势**: 相比Meta Orion采用的SiC波导，玻璃波导材料成本大幅降低
- **可制造性**: 复用已有的规模化生产工艺，无需全新产线投资
- **性能对标**: 70° FoV达到与Orion原型机同等级别，满足沉浸式AR体验需求
- **供应链成熟**: Ray-Ban Display的量产经验可直接迁移至ZOE

### 光学权衡与挑战

- **亮度影响**: 70° FoV下亮度均匀性和总光效可能面临挑战
- **PPD（每度像素）**: 视场角扩大可能降低单位角度的像素密度
- **视觉伪影**: 大视场角几何波导可能引入更明显的鬼影和彩虹效应
- **厚度控制**: 实现70° FoV可能需要增加波导层数，影响整体厚度

### 对Meta AR路线图的影响

- **短期**: Ray-Ban Display（20° FoV）已验证Lumus玻璃波导的量产可行性
- **中期**: ZOE为Meta提供了通往宽FoV消费级AR眼镜的可扩展路径
- **长期**: Meta需在SiC（性能极致）和玻璃（成本可控）之间做出技术路线选择
