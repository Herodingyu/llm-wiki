## SOC 开发流程

- 需求定义 → 架构设计 → RTL → 验证 → 综合 → DFT → 后端 → 流片
- 项目管理工具：JIRA / Confluence / Git / CI/CD

## 芯片项目管理特殊挑战

- 流片周期长（12-18个月）
- 一次性成本高（tape-out）
- 跨团队协作（前端/后端/验证/软件）
- 风险管理（schedule / cost / quality）
