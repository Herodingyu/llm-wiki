# CarPlay Ultra, the next generation of CarPlay, begins rolling out today

## 来源

- **原始文件**: raw/industry/car-infotainment/apple-newsroom-2025-05-carplay-ultra-the-next-.md
- **提取日期**: 2026-05-02

## 摘要

2025年5月15日，Apple正式推出了CarPlay Ultra——下一代CarPlay系统，标志着Apple在车载信息娱乐领域的重大升级。与之前的CarPlay仅投射到中控屏幕不同，CarPlay Ultra实现了与车辆的深度系统级集成，能够控制车内所有驾驶员面向的屏幕，包括仪表盘（Instrument Cluster）。

首发合作伙伴为英国豪华汽车品牌Aston Martin，在美国和加拿大的新车型订单可立即使用CarPlay Ultra，现有配备下一代信息娱乐系统的Aston Martin车型也将在数周内通过经销商软件更新获得支持。CarPlay Ultra不仅将iPhone的地图、媒体等信息与车辆的ADAS、胎压等数据同屏融合显示，还允许车企与Apple设计团队深度合作，为每款车型定制专属的主题设计，保留品牌独特的视觉语言。除Aston Martin外，Hyundai、Kia、Genesis等品牌也已承诺支持CarPlay Ultra。用户需使用iPhone 12或更新机型，并运行iOS 18.5或更高版本。

## 关键要点

### 产品定位与发布信息

| 参数 | 详情 |
|------|------|
| **发布时间** | 2025年5月15日 |
| **首发车企** | Aston Martin（美加市场） |
| **扩展计划** | 未来12个月扩展至全球市场 |
| **承诺车企** | Hyundai、Kia、Genesis等 |
| **设备要求** | iPhone 12或更新机型，iOS 18.5+ |

### 深度集成特性

- **全屏覆盖**: 支持所有驾驶员屏幕，包括仪表盘实时内容
- **仪表组显示**: 速度表、转速表、油量表、温度表等动态数据以精美设计呈现
- **信息融合**: iPhone信息（地图导航、媒体播放）与车辆信息（ADAS状态、胎压监测）同屏显示
- **车辆控制**: 通过屏幕触控、物理按钮或Siri语音管理车辆功能（收音机、空调、性能设置）
- **Widget适配**: iPhone驱动的widget完美适配车载屏幕尺寸和布局

### 定制化设计能力

- **品牌专属主题**: Apple与车企设计团队深度合作，为每款车型量身定制
- **个性化选项**: 驾驶员可自定义颜色主题和壁纸
- **品牌一致性**: 在提供Apple级设计品质的同时，保留车企品牌特色和视觉设计哲学
- **独特体验**: 每款车的CarPlay Ultra界面都独一无二

### 与第一代CarPlay的对比

| 特性 | CarPlay (第一代) | CarPlay Ultra |
|------|-----------------|---------------|
| **屏幕支持** | 仅中控屏幕 | 所有驾驶员屏幕（含仪表组） |
| **车辆集成** | 镜像投射 | 深度系统级集成 |
| **车辆控制** | 有限 | 收音机、空调、性能设置等 |
| **定制能力** | 标准界面 | 车企定制主题 |
| **数据融合** | 仅iPhone数据 | iPhone+车辆数据融合 |
