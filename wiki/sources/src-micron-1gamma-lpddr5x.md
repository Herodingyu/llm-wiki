# Micron Ships World's First 1γ-Based LPDDR5X Sample

## 来源

- **原始文件**: raw/tech/dram/trendforce-news-2025-06-04-news-micron-ships-worlds.md
- **提取日期**: 2026-05-02

## Summary

TrendForce报道，美光（Micron）于2025年6月3日宣布开始出货全球首款基于1γ（1-gamma）工艺的LPDDR5X内存样品，标志着移动内存技术迈入新阶段。1γ是第六代10nm级DRAM工艺（韩国业界称为1c DRAM），线宽约为11-12纳米。这款LPDDR5X提供业界最快的10.7 Gbps传输速率，同时功耗降低20%，封装厚度仅0.61毫米，为业界最薄设计，主要面向2026年旗舰智能手机。美光是首家在移动内存中采用EUV光刻技术的厂商，这一技术优势使其在先进DRAM工艺竞争中取得领先。样品为16GB容量，未来还将提供8GB至32GB全系列容量选项。2025年2月，美光已向Intel和AMD交付1γ DDR5样品，显示该工艺平台同时覆盖移动和PC/服务器市场。在竞争格局方面，SK海力士和三星也在积极推进第六代10nm级DRAM工艺，但美光率先实现LPDDR5X样品出货，占据了时间窗口优势。

## Key Points

### 1. 1γ工艺技术突破
- **工艺代际**：第六代10nm级DRAM（1γ/1c）
- **线宽**：约11-12纳米
- **EUV光刻**：首家在移动内存中采用EUV技术
- **技术优势**：更精细的电路图案，更高的集成度

### 2. LPDDR5X性能指标

| 指标 | 参数 | 对比优势 |
|------|------|----------|
| 传输速率 | 10.7 Gbps | 业界最快 |
| 功耗 | 降低20% | 更长的电池续航 |
| 封装厚度 | 0.61mm | 比竞品薄6%，比前代低14% |
| 容量 | 16GB（样品） | 8GB-32GB全系列规划中 |

### 3. 市场定位与规划
- **目标市场**：2026年旗舰智能手机
- **量产时间**：预计2025年下半年量产爬坡
- **容量规划**：8GB / 12GB / 16GB / 32GB全系列
- **生态合作**：与主流SoC厂商紧密合作优化

### 4. 竞争格局

| 厂商 | 进展 | 时间线 |
|------|------|--------|
| 美光(Micron) | 已出货1γ LPDDR5X样品 | 2025年6月 |
| 美光 | 已交付1γ DDR5样品给Intel/AMD | 2025年2月 |
| SK海力士 | 开发16Gb 1c DDR5，完成量产认证 | 2025年1月 |
| 三星 | 第六代10nm级DRAM目标推迟 | 2025年6月 |

### 5. 美光技术优势
- **唯一性**：唯一同时出货HBM3E和SOCAMM的存储厂商
- **合作**：与NVIDIA合作支持GB300 Grace Blackwell Ultra
- **EUV领先**：首家在移动内存采用EUV，技术储备深厚
- **全系列覆盖**：从移动（LPDDR5X）到PC/服务器（DDR5）全面布局

### 6. 产业意义
- 推动旗舰智能手机内存性能边界
- EUV在DRAM领域的成功应用验证
- 加剧三大DRAM厂商工艺竞争
- 为AI手机、端侧大模型提供硬件基础
