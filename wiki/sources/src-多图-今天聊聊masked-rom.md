## Summary

本文介绍了掩模ROM（Masked ROM）的基本概念、制造原理、结构类型和应用场景。掩模ROM是一种在芯片制造过程中通过光刻步骤直接固化数据的只读存储器，具有极低的生产成本但无法后期修改。文章详细讲解了NOR和NAND两种基本结构的工作原理，以及有源层、接触层、金属层、离子注入四种不同的物理实现技术，并展示了多种实际芯片的显微镜照片。掩模ROM广泛应用于嵌入式系统、微控制器、大批量消费电子和早期游戏卡带中，同时也是我们常说的BootROM的物理实现形式。

## Key Points

### 1. 掩模ROM基本概念与特点

掩模ROM（Masked ROM/编程ROM）是在制造过程中通过光刻步骤直接在芯片上创建逻辑电路物理掩模来实现数据编程的只读存储器。

| 特性 | 说明 |
|------|------|
| **核心优势** | 低成本，无需额外编程步骤或设备 |
| **主要限制** | 制造后数据不可更改，灵活性差 |
| **典型应用** | 嵌入式系统、微控制器、大批量消费电子、早期游戏卡带 |
| **安全关联** | 与PUF、BootROM密切相关 |

### 2. NOR vs NAND 结构

| 结构 | 工作原理 | 特点 |
|------|----------|------|
| **NOR** | 位线拉高到VCC，选择字线时通过晶体管短路到地拉低 | 输出为1，除非任何位线将输出拉低 |
| **NAND** | 开关串联，通过选择非选定行中的开关来激活 | NOR ROM的逻辑反向版本，将开关串联 |

### 3. 四种物理实现技术

| 技术 | 原理 | 安全性 |
|------|------|--------|
| **Active layer（有源层）** | 通过沟道区域控制电流 | 可通过光学方式读取 |
| **Contact layer（接触层）** | 控制晶体管与位线的连接 | 现代CMP工艺需要去层处理 |
| **Metal（金属层）** | 通过金属桥接永久导通晶体管 | 可通过光学方式直接读取 |
| **Implanted（离子注入）** | 改变晶体管阈值电压使其永久关闭 | 提供高水平的安全保护，防止光学读取 |

### 4. 读取技术

掩模ROM内容提取方法包括：
- **电子分析**：研究芯片内部电路结构发现测试模式
- **光学分析**：斜射光照明、共焦显微镜观察植入区域凹陷
- **染色法（Staining）**：使用Dash蚀刻，行业标准方法
- **扫描电容显微镜（SCM）**：测量掺杂引起的电容变化
- **扫描微波阻抗显微镜（SMIM）**：微波阻抗分析
- **能量色散X射线光谱（EDS）**：检测掺杂剂浓度

### 5. 实际应用案例

掩模ROM在实际产品中的典型应用：
- **任天堂NES游戏卡带**：直接存储游戏数据和程序
- **安全处理器配合**：由于易于读取，常与CIC系列安全芯片配合使用防止复制
- **CPU微码**：小型ROM用于构建逻辑阵列
- **RSA SecurID**：1C vs 2C芯片对比展示植入式ROM演进

## Key Quotes

> 掩模ROM（Masked ROM，或称为编程ROM）是一种特殊的只读存储器（ROM），它在制造过程中就被永久性地编程了数据。这种编程是通过在制造过程中的光刻步骤中，直接在芯片上创建逻辑电路的物理掩模来实现的，因此得名"掩模ROM"。

> 掩模ROM的主要优势在于其低成本。由于数据是在芯片制造时直接嵌入的，所以不需要额外的编程步骤或设备，这大大简化了生产过程并减少了生产成本。

> 这些芯片可以直接读取，无需通过图像处理来解码。然而，需要注意的是，由于这些芯片易于读取，因此它们经常与安全处理器配合使用，以防止简单的复制。

> 离子注入ROM的工作原理基本上是从一个具有正常工作晶体管网格的掩模开始。然后，其中一些晶体管会经历额外的轰击，以改变其阈值电压。这种类型的存储器提供了高水平的安全保护，防止光学读取。
