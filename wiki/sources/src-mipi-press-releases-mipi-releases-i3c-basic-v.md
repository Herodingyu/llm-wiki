# MIPI Alliance Releases I3C Basic v1.2 Utility and Control Bus Interface for Mobile, IoT and Data Center Applications

## 来源

- **原始文件**: raw/tech/peripheral/mipi-press-releases-mipi-releases-i3c-basic-v.md
- **提取日期**: 2026-05-02

## 摘要

MIPI联盟发布I3C Basic v1.2版本，作为面向移动、IoT和数据中心应用的实用控制总线接口标准，提供了I3C核心功能的免费开放规范。

## 关键要点

- I3C Basic v1.2正式发布
- 免费提供I3C核心功能规范
- 适用于移动、IoT和数据中心应用
- 促进I3C生态系统的广泛采用

## 技术细节

- **规范版本**: I3C Basic v1.2
- **应用范围**: 移动设备、IoT、数据中心
- **核心功能**: 实用和控制总线接口
- **许可模式**: 免费使用
