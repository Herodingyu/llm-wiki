# Automotive DMS/OMS (Driver/Occupant Monitoring System) Research Report, 2024-2025

## 来源

- **原始文件**: raw/industry/car-infotainment/cn-report-rinc1660089-automotive-dms-oms-dr.md
- **提取日期**: 2026-05-02

## 摘要

ResearchInChina发布的2024-2025年汽车DMS（驾驶员监控系统）/OMS（乘员监控系统）研究报告显示，中国乘用车DMS/OMS市场正处于法规驱动下的高速增长期。2024年1-11月，中国乘用车标配DMS/OMS达328.8万台，安装率16.3%，同比增长39.4%。其中DMS安装量318万台（安装率15.8%，同比增长82.7%），而OMS安装量75.5万台（安装率3.7%，同比下降18.1%），显示出DMS快速普及但OMS增长乏力的分化态势。

法规是推动DMS市场增长的核心驱动力。欧盟GSR II法规要求2024年7月起新车型必须配备ADDW（高级驾驶员分心警告）系统，2026年7月覆盖所有新车；中国C-NCAP 2024版首次将DMS纳入主动安全评估框架，给予2分评分；Euro-NCAP 2026年1月新规进一步扩大了酒驾检测范围。在这些法规推动下，预计2027年中国DMS安装量将达947.3万辆，摄像头DMS市场规模71.1亿元。前十大DMS安装品牌（理想、HIMA、蔚来、极氪、比亚迪等）占据总量的62.7%。

## 关键要点

### 市场规模与增长

| 指标 | 数据 (2024年1-11月) | 同比变化 |
|------|---------------------|----------|
| **DMS/OMS总安装量** | 328.8万台 | +39.4% |
| **总安装率** | 16.3% | - |
| **DMS安装量** | 318万台 | +82.7% |
| **DMS安装率** | 15.8% | - |
| **OMS安装量** | 75.5万台 | -18.1% |
| **OMS安装率** | 3.7% | - |

### 法规驱动因素

| 法规/标准 | 要求 | 生效时间 |
|-----------|------|----------|
| **欧盟GSR II** | 新车型必须配备ADDW | 2024年7月 |
| **欧盟GSR II** | 所有新车必须配备ADDW | 2026年7月 |
| **中国C-NCAP 2024** | DMS纳入主动安全评估，2分 | 已生效 |
| **Euro-NCAP 2026** | 扩大酒驾检测范围 | 2026年1月 |

### DMS vs OMS分化分析

- **DMS高速增长**: 
  - 法规强制要求（EU GSR II、C-NCAP）
  - 技术成熟度高，成本快速下降
  - 与ADAS功能协同价值明确
  
- **OMS增长乏力**:
  - 缺乏强制性法规驱动
  - 隐私顾虑影响消费者接受度
  - 功能价值感知不如DMS直观
  - 但CPD（儿童存在检测）功能正在获得法规关注

### 技术路线演进

- **当前主流**: 视觉感知（摄像头系统）
  - 成熟度高，成本可控
  - 可同时实现DMS和OMS功能
  
- **新兴技术**:
  - **雷达/UWB雷达**: CPD（儿童存在检测）主流方案，不受光线影响
  - **酒精检测传感器**: 车载无意识酒精检测兴起，响应Euro-NCAP新规
  - **直接感知替代间接感知**: UWB/雷达逐步取代基于摄像头的间接推断

- **2027年中国DMS安装量**: 947.3万辆
- **摄像头DMS市场规模**: 71.1亿元
- **趋势**: DMS从"预警"向"干预"演进，与AEB等系统联动
