# Automotive Cockpit SoC Research Report, 2024 - ResearchInChina

## 来源

- **原始文件**: raw/industry/car-infotainment/researchinchina-Htmls-Report-2024-74973html.md
- **提取日期**: 2026-05-02

## 摘要

ResearchInChina发布的2024年汽车座舱SoC研究报告全面剖析了全球及中国座舱SoC市场的竞争格局和技术演进趋势。报告指出，中国本土座舱SoC正在快速崛起，渗透率预计将从2023年的仅4.8%大幅提升至2030年的25%，这意味着未来七年本土厂商将迎来黄金发展期。

当前全球座舱SoC市场呈现"一超几强"格局：Qualcomm凭借SA8155/8295系列在高端市场占据绝对主导地位，已搭载于极越01、新奔驰E级等20余款车型。与此同时，MediaTek与NVIDIA的强强联合推出基于3nm工艺的Dimensity Auto Cockpit平台（CX-1/CY-1/CM-1/CV-1），计划2025年量产，直接挑战高通的高端地位。更值得注意的是，Intel和AMD等PC/服务器芯片巨头正重返汽车座舱市场——AMD V2000A基于7nm Zen 2架构，CPU性能达360-370K DMIPS，已超越高通SA8295P；Intel则基于chiplet技术开发SDV SoC，支持OEM深度定制。

在本土阵营中，芯驰科技（SemiDrive）X9系列出货量已超百万颗并进入QNX BSP列表，芯擎科技（SiEngine）龙鹰一号、杰发科技（AutoChips）以及华为麒麟9610A等已实现大规模量产。日本ASRA联盟（12家车企+5家半导体公司）更是制定了2030年量产chiplet技术SoC的宏大目标，chiplet技术由此成为下一代高性能座舱SoC的重要方向。

## 关键要点

### 中国本土座舱SoC崛起之路

| 年份 | 本土SoC渗透率 | 关键里程碑 |
|------|--------------|------------|
| **2023** | 4.8% | SemiDrive X9出货破百万 |
| **2025E** | ~10% | 龙鹰一号、麒麟9610A规模上车 |
| **2030E** | 25% | 本土SoC成为重要力量 |

### 全球座舱SoC竞争格局

| 厂商 | 代表产品 | 工艺 | CPU算力 | 市场定位 |
|------|----------|------|---------|----------|
| **Qualcomm** | SA8295P | 5nm | 200K+ DMIPS | 高端主导 |
| **Qualcomm** | SA8397 | - | 660K DMIPS | 旗舰 |
| **MediaTek+NVIDIA** | CX-1/CY-1 | 3nm | - | 旗舰挑战者 |
| **AMD** | V2000A | 7nm Zen2 | 360-370K DMIPS | PC级性能 |
| **Intel** | SDV SoC | Chiplet | 可定制 | OEM定制 |
| **SemiDrive** | X9系列 | - | - | 本土量产 |
| **SiEngine** | 龙鹰一号 | - | - | 本土中高端 |

### 各阵营战略分析

**Qualcomm阵营（一超）**:
- 市场地位: 高端座舱SoC绝对领导者
- 产品节奏: SA8155→8295→8397，持续迭代
- 生态壁垒: 与QNX、Android Automotive深度绑定
- 挑战: 面临MediaTek+NVIDIA和AMD的直接竞争

**MediaTek+NVIDIA阵营（强强联手）**:
- 合作模式: MediaTek SoC集成NVIDIA GPU/AI
- 核心优势: 3nm先进工艺 + RTX图形 + AI算力
- 目标市场: 2025年量产，冲击高端
- 差异化: 端侧LLM + 光线追踪游戏

**PC巨头重返（Intel/AMD）**:
- AMD优势: Zen 2架构CPU性能超越高通
- Intel优势: Chiplet技术，x86生态
- 策略: 以性能优势切入，争夺高端市场
- 挑战: 车规认证、功耗控制、生态适配

**本土厂商崛起**:
- SemiDrive: X9系列出货破百万，QNX认证
- SiEngine: 龙鹰一号，7屏支持
- 华为: 麒麟9610A + Harmony OS生态
- 挑战: 高端市场突破、品牌认知度

### Chiplet技术趋势

- **Intel SDV SoC**: 基于chiplet支持OEM定制化
- **Renesas R-Car X5H**: UCle标准chiplet扩展
- **日本ASRA联盟**: 12家车企+5家半导体，目标2030年量产
- **价值**: 灵活组合计算/图形/AI模块，降低成本，延长产品生命周期
