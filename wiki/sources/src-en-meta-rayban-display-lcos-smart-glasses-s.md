# Meta Launches 'Ray-Ban Display' with LCoS Smart Glasses, Showcases PIC-Based LCoS at SID 2025

## 来源

- **原始文件**: raw/industry/smart-glasses/en-meta-rayban-display-lcos-smart-glasses-s.md
- **提取日期**: 2026-05-02

## 摘要

2025年9月18日（当地时间17日），Meta在Meta Connect 2025大会上正式发布了其首款带显示屏的消费级智能眼镜——Meta Ray-Ban Display。该产品战略定位于现有Ray-Ban AI音频眼镜与2024年展示的Orion AR原型眼镜之间的过渡形态，预计于2025年9月底在美国市场上市。

产品核心显示方案采用OmniVision的单面板全彩LCoS微显示器，规格为600×600分辨率、42 PPD（每度像素）、20°单眼视场角，峰值亮度高达5,000 nits，配合Lumus波导技术实现户外环境下的清晰可视。这些规格精准满足了信息型AR设备的技术要求（20-35° FoV、高亮度、低功耗）。Meta选择LCoS而非LEDoS是基于技术成熟度、功耗效率和全彩实现能力的综合考量。值得注意的是，Meta在同期SID 2025上还展示了基于光子集成电路（PIC）的超紧凑激光微显示器研究成果，有望将AR光引擎体积缩小至1cm³以下，展现了下一代显示技术的演进方向。

## 关键要点

### 产品规格与显示技术

| 参数 | 规格 |
|------|------|
| **显示器类型** | OmniVision单面板全彩LCoS |
| **分辨率** | 600×600 |
| **PPD** | 42 pixels per degree |
| **视场角** | 20° 单眼 |
| **峰值亮度** | 5,000 nits |
| **波导技术** | Lumus波导 |
| **显示位置** | 右镜片底部单目显示 |

### 技术路线选择逻辑

- **LCoS vs LEDoS**: Meta选择LCoS而非LEDoS，核心考量包括：
  - 技术成熟度更高，供应链更完善
  - 全彩实现能力更强
  - 功耗效率更优，有利于延长续航
- **信息型AR定位**: 20° FOV满足信息提示、导航、通知等场景需求，而非沉浸式AR体验
- **户外可用性**: 5,000 nits高亮度确保在阳光直射下仍可清晰阅读显示内容

### SID 2025研究前沿：PIC-based LCoS

- **光子集成电路（PIC）**: 将传统LCoS所需的光学元件（偏振分束器PBS、聚焦透镜、二色镜）集成到单芯片上
- **体积突破**: AR光引擎可缩小至**1cm³以下**
- **性能提升**: 50度视场角，高色均匀性
- **平台潜力**: PIC-based激光照明具备规模化平台潜力，有望成为下一代AR显示技术支柱

### 产业生态影响

- **供应链**: OmniVision、Himax Display、Avegant、Raontec等厂商正推进下一代高亮度、高对比度LCoS引擎研发
- **市场时间窗**: LEDoS预计2028年后才具备全面竞争力，此前全彩LCoS将是AR眼镜市场的关键解决方案
- **Meta双轨策略**: 同步推进商用产品（OmniVision LCoS）和前沿研究（PIC-based微显示），加速AR产业生态发展
