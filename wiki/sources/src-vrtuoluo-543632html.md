# 两万字汇总｜2025光博会，看到AR产业全面商业化爆发的曙光

## 来源

- **原始文件**: raw/industry/smart-glasses/vrtuoluo-543632html.md
- **提取日期**: 2026-05-02

## 摘要

2025年CIOE（中国国际光电博览会）汇集了歌尔光学、JBD、鸿石智能、广纳四维等数十家XR产业链核心厂商，全面展示了AR光学技术的最新进展和产业化成果。本次展会反映出AR产业正处于从实验室走向大规模商业化的关键转折点。

AR光学领域呈现出明显的技术多元化趋势：三大光学方案（表面浮雕光栅SRG、体全息PVG/VHG、几何光波导）并行发展；三大制造工艺（纳米压印、刻蚀、DUV+刻蚀）各显神通；三种基底材料（树脂、玻璃、碳化硅SiC）针对不同应用场景展开竞争。其中，纳米压印仍是当前市场主流工艺，但刻蚀工艺凭借其更高的加工精度和材料适应性，正成为行业公认的未来方向。歌尔光学推出的SiC刻蚀全彩衍射光波导F30Se将厚度压缩至0.65mm、重量仅3.5g，刷新了行业轻薄纪录。

在显示技术方面，Micro-LED需求呈现翻倍增长态势，全彩X-Cube合色方案成为短期技术共识。JBD已完成全年销售目标，合肥生产线面板产能达到单月100万片。与此同时，PVG（偏振体全息）体全息波导凭借成本优势（谷东自研PVG每片仅150-200元）开始扮演"价格屠夫"角色，有望加速AR眼镜的大众化普及。

## 关键要点

### AR光学技术格局

| 维度 | 技术路线 | 代表厂商/产品 |
|------|----------|---------------|
| **光学方案** | 表面浮雕光栅(SRG) | 歌尔、至格、立讯 |
| | 体全息(PVG/VHG) | 谷东、形意智能 |
| | 几何光波导 | Lumus、Optinvent |
| **制造工艺** | 纳米压印 | 当前主流，成本低 |
| | 刻蚀 | 高精度，未来趋势 |
| | DUV+刻蚀 | 超高精度，上亿投入 |
| **基底材料** | 树脂 | 轻薄，成本低 |
| | 玻璃 | 成熟工艺，光效好 |
| | 碳化硅(SiC) | 折射率2.7，极致性能 |

### 关键产品与技术突破

- **歌尔F30Se**: SiC刻蚀全彩衍射光波导，0.65mm厚度，3.5g重量，30° FoV，行业轻薄新标杆
- **广纳四维C45C**: 碳化硅全彩刻蚀波导，0.7mm厚度，3.7g重量，800nits/lm亮度
- **至格SID25A-ME**: 全彩Micro-LED+衍射光波导，25° FoV，4000nits亮度，0.8mm厚度
- **谷东H2030C**: 全彩PVG波导模组，DLP光机，30° FoV，入眼亮度>1200nit
- **形意智能AR99**: VHG波导+Micro-LED，售价仅999元，极致性价比

### 显示技术产业化进展

- **Micro-LED**: JBD蜂鸟II（0.2cc体积，0.5g重量，500×380分辨率，480Hz刷新率），合肥产线月产能100万片
- **全彩化方案**: X-Cube合色成为短期共识，鸿石极光耀影XC6（0.35cc光机，250万尼特亮度）
- **Micro-OLED**: 厂商积极寻求AI AR眼镜等新场景突破，但面临LCoS和Micro-LED竞争压力
- **LCoS**: 光峰科技LCoS+谷东PVG一拖二波导方案，终端价1500-2000元

### 产业链趋势

- **一拖二波导**: 单光机驱动双目显示，节省BOM成本，但牺牲3D显示能力
- **刻蚀工艺布局**: 歌尔、AAC等龙头已布局刻蚀工艺，AAC收购Dispelix瞄准单片全彩SRG波导
- **国产主控芯片**: 恒玄2800、全志V821等国产芯片崛起，挑战高通垄断
- **眼镜支付**: 雷鸟V3、小米AI眼镜已支持，蚂蚁集团为唯一技术供应商
