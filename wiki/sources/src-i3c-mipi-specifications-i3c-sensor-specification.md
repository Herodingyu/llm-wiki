# I3C and I3C Basic | MIPI

## 来源

- **原始文件**: raw/tech/peripheral/I3C-mipi-specifications-i3c-sensor-specification.md
- **提取日期**: 2026-05-02

## 摘要

MIPI联盟官方I3C规范页面，介绍了I3C和I3C Basic标准规范，包括传感器规格和协议细节，是I3C开发的权威参考。

## 关键要点

- MIPI I3C是MIPI Alliance管理的开放标准
- I3C Basic是免费提供的子集规范
- 定义了传感器接口的标准化命令集
- 支持多种数据传输模式和速率

## 技术细节

- **规范版本**: I3C v1.x / I3C Basic
- **传感器规范**: MIPI I3C Sensor Specification
- **核心特性**: 动态寻址、带内中断、HDR模式
- **标准化命令**: CCC (Common Command Code)
